Apkizer Setup for AAD

this zip file. Extract it and move the zip file to a place where you won't delete it (Do not save to your Android SDK folder.)
Open a terminal window. Use command + space or click on the search icon



Navigate to the folder you extracted. HINT: type "cd " and then drag the folder to the window to go there instantly
Type in: "chmod +x ./install.sh" (without the quotes)

Then, type in ./install.sh.

Now open the build.gradle.ftl file in textedit. Replace ##USERNAME## with the user name you use to log on to this computer.

Now, go to where you installed Android Studio. On Macs, this is in the executeable itself. (Right Click > Show Package Contents)



Navigate to "contents/plugins/android/lib/templates/gradle-projects/NewAndroidModule/root" and replace the "build.gradle.ftl" file with the "build.gradle.ftl" file included in the zip file.
This applies to all new projects only. Run the task by clicking on the "Gradle" sidebar, click on the drop downs and double click archonDebug.

If you don't see the bar, go to View > Tool Windows > Gradle
